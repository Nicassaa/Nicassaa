Lesson 4: CSS Class

In this lesson, I learned about CSS Classes, which are used to style specific parts of a webpage.

A class lets us apply the same design to many elements without repeating the code.

It helps make web pages more organized, consistent, and easy to update.

I also learned how to create, call, and combine classes in HTML and CSS to make my designs neat and creative.

Learnings:

  • It explains what classes are in CSS — reusable styles that you can apply to many elements.
  • It shows how to create a class using a dot (.) like .highlight {color: green;}.
  • It teaches how to apply a class to HTML using class="classname".
  • It also covers element + class selectors (like p.note {color: blue;}), which target only a specific element type.
  • Lastly, it mentions grouping selectors — combining multiple selectors with commas to make your CSS shorter and cleaner.
