Lesson 5: CSS SELECTORS

In this lesson, I learned about CSS Selectors, which are used to choose or “select” parts of an HTML page that we want to design.

Selectors help us apply different styles like color, font, or background to specific elements.

There are many kinds of selectors such as class, ID, element, universal, group, and attribute selectors, and each one has its own purpose.

Understanding CSS selectors makes it easier to create clean, organized, and creative web designs.

Learnings:

  • I learned that CSS selectors are used to choose which parts of an HTML page we want to design or style.
  • The lesson taught me about different types of selectors like class, element, ID, universal, group, and attribute.
  • I understood that the class selector can be used many times, while the ID selector is for unique elements only.
  • The element selector styles all of one type of tag (like all

    tags), and the universal selector styles everything on the page.

  • I also learned that group selectors save time by styling multiple elements together, and attribute selectors let us style tags based on their attributes.
  • Overall, this lesson helped me realize how selectors make CSS powerful and organized, helping web designers easily control how every part of a webpage looks.
