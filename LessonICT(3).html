Learnings:

  • I learned that the <span> tag is used to style small parts of text inside a line, like changing the color or font of one word.
  • The <div> tag is used to group elements together to create sections or boxes on a webpage.
  • I realized that both tags help make web pages organized and easy to design using CSS.
  • Using <span> and <div> makes my webpage look neater, colorful, and well-arranged.
  • I now understand that <div> works like a container, while <span> is for inline styling within text.
