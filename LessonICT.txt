  <strong style="font-size:24px;">Learnings:</strong>
<li> CSS is a very helpful tool in web design because it makes webpages look neat, colorful, and organized.</li>
<li>Its capability is to change the style of a website, like the colors, fonts, and layout, without changing the main content. </li>
<li>One advantage of using CSS is that it saves time, you can style many pages at once. The benefit is that it helps make websites more attractive and easier to read. </li>
<li>In creating a webpage, the presentation layer shows the design using CSS, the behavior layer adds actions using JavaScript, and the content layer holds the text and images using HTML. </li>
<li> CSS can be used to make websites look professional and creative. It allows us to express our ideas and make information more interesting to others.</li>
